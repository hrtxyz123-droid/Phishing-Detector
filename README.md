# 🛡️ Phishing Detector Pro

An advanced Chrome Extension that detects phishing attempts in real-time using a hybrid AI approach.

## 🚀 How to Run
1. **Backend:** Navigate to the backend files, run `npm install`, then `node server.js`.
2. **Frontend:** Run `npm install` in the root, then `npx webpack`.
3. **Chrome:** Load the `dist` folder as an "Unpacked Extension".

## 🧠 Technical Specs
* **AI Model:** 500-character input window with Sigmoid activation.
* **Backend:** Secure IMAP fetching via Node.js.
* **APIs:** Google Safe Browsing.
